- J'ai codé la lecture du fichier CSV dans la classe GestionCSV.java.

- J'ai développé la classe ValidationUtils.java pour vérifier les données.
  Elle permet de s'assurer que les notes sont valides et que les données du fichier
  ne contiennent pas d'erreurs.
