# Gestion des notes des étudiants – Projet Java

## Description

Ce projet Java a pour objectif de gérer les notes des étudiants à partir d’un fichier CSV.  
Le programme permet de lire les données, calculer les moyennes, classer les étudiants et générer un fichier de résultats.


## Fonctionnalités

- Lecture des données depuis un fichier CSV
- Calcul de la moyenne des étudiants
- Classement des étudiants par performance
- Génération d’un fichier de résultats
- Validation des données

## Exécution du programme

1. Ouvrir le projet dans IntelliJ IDEA
2. Vérifier que le fichier `etudiants.csv` est présent dans le dossier `data`
3. Exécuter la classe `Main.java`
4. Consulter le fichier `resultats.csv` généré
